# PyPosts
- PyPosts is a framework that can be used as CMS (Content Management System)
- This framework supports Adding, Removing, Updating, and Reading Posts
- This framework uses MySQL for it's data storage

## Dependencies
- Server
	- MySQL Server
		- **Local Hosted** (Please configure tunnelling if you only have remote server)
- Python modules/libraries (not built-in, installed via `pip`)
	- `mysql-connector`

- Python modules/libraries (built-in)
	- `sys`
	- `datetime`
	- `json`
